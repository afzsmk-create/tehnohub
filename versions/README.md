# Архив версий калькулятора гибки

## Актуальные файлы (в корне репозитория)
- `index.html` — главная страница технохаба
- `bender-3d.html` — **✅ СТАБИЛЬНАЯ ВЕРСИЯ** — 3D калькулятор гибки (Three.js)

## Bender (оригинал)
- `Bender.html` — исходный калькулятор с 3 пресетами (уголок, П, Z)

## Multibender (динамические гибы)
- `Bender_v2.html` — добавлены динамические гибы (произвольное количество)
- `Bender_v3.html` — material presets, новые пресеты (Короб, Шляпный, Швеллер)
- `Bender_v4.html` — фикс слайдеров, исправлены направления пресетов
- `Bender_v5.html` — удалены лишние пресеты, подсказка перенесена вниз
- `Bender_v6.html` — фикс: первая полка всегда горизонтальна

## Multibender 3D (Three.js) — архив
- `multibender_3d_v1_gemini.html` — первая 3D версия (от Gemini)
- `bender-3d_v1.html` — вторая версия 3D
- `multibender.html` — 2D версия с динамическими гибами

## Текущая стабильная версия: `bender-3d.html`

### Возможности:
- 3D визуализация детали (Three.js, OrbitControls)
- Деталь стоит на торце
- Кнопки вида: Изометрия, Сверху, Сбоку, Дом
- Плавные переходы камеры
- Пресеты: Уголок, П-образный, Z-зигзаг, Короб, Шляпный, Швеллер
- Пресеты материалов: Сталь, Нерж., Алюм., Медь
- Динамические гибы с направлением (вверх/вниз)
- Пресеты углов: 60°, 90°, 120°, 150°
- Расчёт: BA, BD, OSSB, K-фактор, усилие пресса
- Развёртка с нетто-размерами и BA
- Экспорт в DXF
- Сохранение/загрузка профилей (localStorage)
- Справка (кнопка ?)
- Пуансон с носиком
